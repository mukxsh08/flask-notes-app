# 📝 Flask Notes Management System

A secure and simple Notes Management Web Application built using **Flask**.  
Users can create, view, update, and delete notes with authentication support.

---

## 📌 Features

- 🔐 User Authentication (Login & Signup)
- 📝 Create, Read, Update, Delete Notes (CRUD)
- 🔒 Secure Password Hashing
- 💾 SQLite Database Integration
- ⚡ Lightweight & Fast Flask Backend
- 🎯 Clean UI with Dashboard

---

## 🛠️ Tech Stack

- **Backend:** Python, Flask
- **Frontend:** HTML, CSS, Bootstrap
- **Database:** SQLite
- **Others:** Jinja2, Werkzeug (for security)
